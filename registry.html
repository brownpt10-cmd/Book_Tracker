Project Registry

The "what exists" map of Pat's Claude ecosystem — every runnable task, project, and supporting doc across Cowork, Claude Code, and Claude Projects (Chat), grouped by life category. Pair it with DASHBOARD.md (the "what needs attention today" digest).

Why this is hand-maintained: no single Claude view spans Chat + Cowork + Code today, so nothing can enumerate this automatically. Source of truth is InvestmentTracker/Claude Registry.xlsx — edit that, then run "update the registry" to re-render this file and the cards.

System tags: Cowork (scheduled/manual Cowork run) · Code (Claude Code project) · Project (Claude Project + skills, run from Chat) · Reference (a doc that feeds other tasks, not a runnable itself).

Update cadence: manual. Running "update the registry" also rebuilds the HTML (python3 build_dashboard.py) so the rendered page stays in sync.
